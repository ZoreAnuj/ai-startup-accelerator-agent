# AI Startup Accelerator Agent

This project is a multi-agent system built with CrewAI that automates the early-stage startup development process. It explores how autonomous AI agents can collaborate to transform a raw idea into a structured plan and MVP prototype.

## Key Features
*   Multi-agent collaboration for ideation, validation, and planning.
*   Automated generation of business plans, user stories, and technical specifications.
*   Integration with Groq's LLM API for high-speed inference.
*   Modular design allowing customization of agent roles and workflows.

## Tech Stack
*   CrewAI
*   LangChain
*   Groq API
*   Python

## Getting Started
1.  Clone the repo: `git clone https://github.com/zoreanuj/ai-startup-accelerator-agent.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Set your Groq API key: `export GROQ_API_KEY='your-key-here'`
4.  Run the main script: `python main.py`
